# XAY-DUNG-WEB-BAN-HANG-MY-PHAM-SKINCARE
Đề tài “Xây dựng website bán hàng mỹ phẩm SkinCare” tạo trang web thương mại điện tử cho phép người dùng xem, tìm kiếm và mua sản phẩm chăm sóc da. Hệ thống gồm hai vai trò: người dùng (mua hàng, thanh toán, theo dõi đơn) và quản trị viên (quản lý sản phẩm, đơn hàng, người dùng, thống kê doanh thu).
